# Baklafy

Baklafy is a music application built with Node.js, MongoDB, Socket.IO, and modern web technologies. The project focuses on providing a simple way to search, download, organize, and listen to music while exploring real-world software engineering concepts such as authentication, real-time communication, database management, and scalable backend architecture.

---

## Features

### 🎶 Music Management
- Search for songs
- Download songs as MP3 files
- Create and manage playlists
- Offline playback support

### 👤 Account System
- User accounts
- Secure password hashing with bcrypt
- Persistent login sessions
- Session token management

### 📂 Playlist Support
- Custom playlist creation
- Playlist storage using MongoDB
- Playlist data management

### 🌐 Real-Time Communication
- Socket.IO client-server communication
- Real-time download updates
- Live account verification

### 💾 Offline Functionality
- Access downloaded songs without an internet connection
- Persistent session handling
- Local storage integration
